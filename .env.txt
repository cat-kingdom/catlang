# ----------------------------------------------------------------------------
# OpenRouter Configuration (Alternative to OpenAI)
# ----------------------------------------------------------------------------
# OpenRouter provides access to multiple LLM models including Grok
# Get your API key from: https://openrouter.ai/keys
# 
# To use OpenRouter with Grok:
# 1. Set LLM_PROVIDER=openai (OpenRouter uses OpenAI-compatible API)
# 2. Set OPENAI_API_KEY to your OpenRouter API key
# 3. Set OPENAI_BASE_URL=https://openrouter.ai/api/v1
# 4. Set OPENAI_MODEL to a Grok model (see below)
#
# Example Grok models available on OpenRouter:
# - x-ai/grok-beta (Grok Beta)
# - x-ai/grok-2-vision-1212 (Grok 2 Vision)
# - x-ai/grok-3 (Grok 3)
# - x-ai/grok-4 (Grok 4)
# - x-ai/grok-4.1-fast (Grok 4.1 Fast)
# - x-ai/grok-vision-beta (Grok Vision Beta)
#
# Example configuration for Grok via OpenRouter:
# LLM_PROVIDER=openai
# OPENAI_API_KEY=sk-or-v1-your-openrouter-api-key-here
# OPENAI_BASE_URL=https://openrouter.ai/api/v1
# OPENAI_MODEL=x-ai/grok-beta
